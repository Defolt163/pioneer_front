# PIONEER — Next.js Frontend

## Запуск

```bash
npm install
npm run dev
```

Открой: http://localhost:3000

## Экраны

| Маршрут | Экран | По ТЗ |
|---|---|---|
| `/` | → редирект на `/select-role` | — |
| `/select-role` | Выбор типа пользователя | Раздел 10.1 |
| `/register` | Регистрация (телефон + SMS) | Раздел 10.2 |
| `/services` | Выбор услуги (заглушка) | Раздел 10.3 |

## Структура файлов

```
pioneer/
├── app/                         # Next.js App Router
│   ├── globals.css              # стили, CSS-переменные, мобильный фрейм
│   ├── layout.jsx               # корневой layout
│   ├── page.jsx                 # / → /select-role
│   ├── select-role/
│   │   └── page.jsx             # экран выбора типа пользователя (ТЗ 10.1)
│   ├── register/
│   │   └── page.jsx             # регистрация по телефону + SMS (ТЗ 10.2)
│   └── services/
│       └── page.jsx             # выбор услуги — заглушка (ТЗ 10.3)
│
├── components/
│   └── ui/
│       ├── Button.jsx           # кнопка
│       ├── PhoneInput.jsx       # ввод телефона с маской
│       ├── SmsInput.jsx         # ввод SMS-кода
│       └── TopBar.jsx           # шапка с кнопками "Назад" и "Выход"
│
├── hooks/                       # (пусто, для будущей логики)
├── lib/
│   └── constants.js             # роли, маршруты
│
└── package.json
```
